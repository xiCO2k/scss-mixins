# xico2k-scss-mixins

This is my personal mixins.

## Installation
With [npm](https://www.npmjs.com):
```
$ npm i xico2k-scss-mixins
```
or with [yarn](https://yarnpkg.com):
```
$ yarn add xico2k-scss-mixins
```

## Usage

On your SCSS file just add:
```css
@import "~xico2k-scss-mixins/mixins";
```
